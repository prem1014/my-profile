Angular is one of the most popular javascript frameworks for creating web apps. There is a huge difference between Angular Js(1.x) and Angular 2. Angular team has re-written the Angular and also decided to call Angular instead of calling Angular 2 or Angular 4 or any latest version.

Angular Set Up

You need to set up your development environment before you can do anything. Install Node.jsÂ® and npm if they are not already on your machine.
Verify that you are running at least node 6.9.x and npm 3.x.x by running node -v and npm -v in a terminal/console window. Older versions produce errors, but newer versions are fine.
Install Angular CLI globally on your machine. Open your terminal window and run below command.
npm install -g @angular/cli
Create a new project using Angular CLI. Run the below command in your terminal window. Wait for few minutes as it will take time to set up the project.

//it will create project with .css file

ng new your-app-name

//If you want to create project with scss/sass/less, then use below command

ng new your-app-name --style=scss or sass or lees
Once you done with your project set up, go to project directory and start the application by running below command.
cd your-app-name
ng serve --open
The ng serve will launch the servers and --open or -o flag will open your browser on http://localhost:4200/. The ng serve command will watch your files and rebuild apps as you make any changes to those files.
Your broser will display below message.
Now you are ready with your set up and you can start building your own apps.
